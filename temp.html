Filters↑
Click the filter icon on the sidebar to access the data filters. The options on the Selectors and Indicators panels can be used to screen the plots on the map (see below).
Selectors↑
The Selectors panel is shown by default. Selectors are categorical variables such as land-cover type, Ecological Site ID, and state. Choose a selector by clicking the following elements in the given order:
  1. The Selectors tab (open by default)
  2. A selector button in the upper part of the Selectors panel
  3. The corresponding dropdown that appears in the lower part of the Selectors panel
  4. A subsequently displayed dropdown option
  5. The dropdown/autocomplete input for the desired sub-selector (for the Date selector, see below)
Next, begin typing in the autocomplete input field. Select the desired field from the dropdown list, click the Add/Remove button to create a selection, and then click the Submit button. Available selectors are described below.
Land Cover
Select a land-cover type using the following options:
  • MODIS IGBP Name
  • MODIS IGBP Value
Ecoregion
Select an ecoregion using the following options:
  • Ecoregion Level I
  • Ecoregion Level II
  • Ecoregion Level III
  • Ecoregion Level IV
MLRA
Select an MLRA using the following options:
  • MLRA Name
  • MLRA Symbol
Ecological Site
Select a type of ecological site using the following options:
  • Ecological Site ID
  • MLRA Prefix (from Ecological Site ID)
  • State
State
Select a state using the following option:
  • US State
Currently, only US states are available.
Project
Select a project using the following option:
  • Project Name
Date
To select a date range, either (a) drag the range-slider handles or (b) enter dates in the input boxes; use the following range-slider option:
  • Date Visited
Data Type
Select a data type (i.e., retrieve only database records that include entries for the specified data type that are not NULL) using the following option:
  • Field Method
Indicators↑
As described in the second edition of the Monitoring Manual, indicators include plot measurements such as line-point intercept (e.g., percent bare ground) and vegetation height (e.g., mean woody height) Select data ranges using the dropdown range sliders. Choose an indicator by clicking the following elements in the given order:
  1. The Indicators tab
  2. An indicator button in the upper part of the Indicators panel
  3. The corresponding dropdown that appears in the lower part of the Indicators panel
  4. A subsequently displayed dropdown option
To select an indicator range, either (a) drag the range-slider handles or (b) enter indicator values in the input boxes. Note that each range slider is preset, with the lower handle/value corresponding to the mean minus three standard deviations, and the upper handle/value corresponding to the mean plus three standard deviations. Available indicators are described below.
Line-Point Intercept
Select line-point intercept data ranges using the following indicators:
  • Percent Bare Ground
  • Percent Foliar Cover
  • Percent Litter Cover
  • Percent Rock Fragment Cover
  • Percent Lichen Cover
  • Percent Cyanobacteria Cover
Vegetation Height
Select vegetation-height data ranges using the following indicators:
  • Mean Woody Height
  • Mean Herbaceous Height
Canopy Gap
Select canopy gap data ranges using the following indicators:
  • Canopy Gaps 25-50 cm (%)
  • Canopy Gaps 51-100 cm (%)
  • Canopy Gaps 101-200 cm (%)
  • Canopy Gaps > 200 cm (%)
Soil Stability
Select soil-stability data ranges using the following indicators:
  • Mean Soil Stability
  • Mean Soil Stability: Protected Samples
  • Mean Soil Stability: Unprotected Samples
Control Buttons↑
Submit Button
Click the Submit button to
  1. Select all plots on the map that satisfy the chosen filter conditions
  2. Retrieve and populate the tables (that appear below the map) with data that satisfy the chosen filter conditions
Clear Button
Click the Clear button to
  1. Deselect all selected plots on the map
  2. Remove the dynamically generated tables below the map
  3. Return the map layer to the default (Google Terrain)
  4. Uncheck the Select features from polygon checkbox in the Filters panel
  5. Remove all chosen selectors and indicators from the Filters panel
Checkbox↑
Clicking the Submit button with the Select features from polygon checkbox checked will (a) show the intersection of filtered plots and polygon-selected plots on the map and (b) show the data for the intersection in the tables that appear below the map. If no polygon selections exist but filters have been chosen, only the filtered plots (and data) are returned. If no filters have been selected but polygon selections exist, the polygon-selected plots and data are returned. Note that choosing additional filters and adding polygons with the Select features from polygon checkbox checked will continue to return the intersection of the filtered plots and polygon-selected plots. Clicking either the Clear button in the Filters panel or the Reset button on the map will uncheck the Select features from polygon checkbox.
Map Controls↑
There are 5 controls on the right-hand side of the map:
  1. A zoom (+/-) control
  2. A zoom slider
  3. A polygon selector/deselector
  4. A map-layer selector
  5. A reset button
Zoom (+/-) Control↑
The zoom buttons allow users to incrementally zoom in and out by clicking the plus (+) button and the minus (-) button, respectively.
Zoom Slider↑
Use the zoom slider to quickly zoom in and out.
Polygon Selector/Deselector↑
Clicking the polygon button (the button with the polygon icon ) produces a submenu with Select and Deselect options. Drawing a polygon with the Select option adds plots to the total set of selected plots (polygon selections are additive). Drawing a polygon with the Deselect option removes plots from the selection set.
Map-Layer Selector↑
Clicking the map-layer button (the button with the map-layers icon ) produces a menu with 4 Google map-layer options:
  1. Terrain (the default)
  2. Satellite
  3. Road
  4. Hybrid
Clicking the Clear button on the Filters sidebar panel or the Reset button on the map will reset the map layer to Google Terrain.
Reset Button↑
Click the Reset button to do the following:
  1. Deselect all selected plots on the map
  2. Remove the dynamically generated tables below the map
  3. Return the map layer to the default (Google Terrain)
  4. Uncheck the Select features from polygon checkbox on the Filters panel
Note that the Reset button acts like the Clear button on the Filters panel — with one important difference: the Reset button does not deselect chosen selectors and indicators.
Map Annotations↑
There are three map annotations:
  1. A scale line
  2. A Selected Plots annotation
  3. A Latitude/Longitude annotation
Scale Line↑
The scale line, located in the lower right-hand corner of the map, provides a metric distance scale that changes according to zoom level.
Selected Plots↑
The Selected Plots subsection (below the map on the left) shows the number of plots that have been selected using the polygon map control or the selectors and indicators in the Filters panel.
Latitude/Longitude↑
The Latitude/Longitude subsection below the map on the right-hand side shows the latitude and longitude values that correspond to the current position of the mouse pointer when hovering over the map.
Tables↑
The tables that appear below the map when plots are selected are as follows:
  1. Filters
  2. Indicators
  3. Species Indicators
  4. Line-Point Intercept
  5. Canopy Gap
  6. Height
  7. Species Inventory
  8. Soil Stability
The Filters and Indicators tables are always shown, but the other 6 tables are only populated with data if the user has logged in (and so has agreed to the LDC data disclaimer). It's also important to note that while all selected sites are shown on the map, the corresponding data rows in the tables are limited to (a) 10 rows each for the Filters and Indicators tables and (b) 200 rows for each of the other 6 tables. Pressing the Next button that appears at the bottom of each table will advance to the next set of data rows for a given table. Use the horizontal scroll bar at the bottom of the tables and the vertical scroll bar on the right-hand side of the tables to view rows and columns.
Filters↑
This table shows the data, 10 rows at a time, corresponding to the selectors and indicators chosen in the Filters panel.
Indicators↑
This table shows additional indicator data, 10 rows at a time, corresponding to the selectors and indicators chosen in the Filters panel.
Species Indicators↑
The Species Indicators table shows all species-indicators data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
Line-Point Intercept↑
The Line-Point Intercept table shows the raw line-point intercept data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
Canopy Gap↑
The Canopy Gap table shows the raw canopy gap data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
Vegetation Height↑
The Vegetation Height table shows the raw vegetation height data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
Species Inventory↑
The Species Inventory table shows the raw species inventory data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
Soil Stability↑
The Soil Stability table shows the raw soil stability data, 200 rows at a time, that correspond to the plot Primary Key values in the Filters table.
